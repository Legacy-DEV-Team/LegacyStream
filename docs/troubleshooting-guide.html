Quick Diagnostics

System Check

Run these quick checks to identify common issues:

  1. Check Server Status: Verify LegacyStream is running
  2. Port Availability: Ensure ports 8000, 8001 are not in use
  3. Firewall Settings: Check Windows Firewall configuration
  4. Network Connectivity: Test internet connection
  5. Audio Devices: Verify audio hardware is working

Common Error Messages

  • "Port already in use": Another application is using the port
  • "Access denied": Administrator privileges required
  • "SSL certificate error": Certificate configuration issue
  • "Stream not found": Mount point configuration problem
  • "Connection refused": Network or firewall issue

Installation Issues

Installer Won't Start

  • Check if you have administrator privileges
  • Verify the installer file is not corrupted
  • Disable antivirus temporarily
  • Try running in compatibility mode

Missing Dependencies

If you encounter missing DLL errors:

  • Install Visual C++ Redistributable
  • Update Windows to latest version
  • Install .NET Framework if required
  • Check system requirements

Installation Fails

  • Free up disk space (minimum 100 MB required)
  • Close other applications
  • Run installer as administrator
  • Check Windows Event Viewer for errors

Startup Problems

Application Won't Launch

  • Check if all required files are present
  • Verify Qt libraries are accessible
  • Run as administrator
  • Check Windows Event Viewer

Server Won't Start

  • Check if ports are already in use: 
    netstat -an | findstr :8000
netstat -an | findstr :8001
  • Verify configuration file syntax
  • Check administrator privileges
  • Review error logs

Configuration Errors

  • Validate INI file format
  • Check file paths and permissions
  • Verify SSL certificate paths
  • Test configuration with default settings

Log Analysis

Log File Locations

  • Server Log: logs/server.log
  • Error Log: logs/error.log
  • Access Log: logs/access.log
  • Debug Log: logs/debug.log

Common Log Entries

# Server startup
[INFO] LegacyStream server starting...
[INFO] Loading configuration from config.ini
[INFO] SSL certificate loaded successfully
[INFO] Server started on port 8000

# Error examples
[ERROR] Failed to bind port 8000: Address already in use
[ERROR] SSL certificate not found: certs/server.crt
[ERROR] Stream mount point /live not found
[ERROR] Audio codec not supported: invalid_codec

Debugging Commands

  • View recent logs: tail -f logs/server.log
  • Check for errors: grep ERROR logs/error.log
  • Monitor access: tail -f logs/access.log
  • Debug mode: Enable debug logging in configuration
