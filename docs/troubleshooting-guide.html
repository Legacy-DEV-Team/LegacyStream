1. Quick Diagnostics

1.1 System Health Check

Run these quick checks to identify common issues:

Check Server Status

# Check if LegacyStream is running tasklist /FI "IMAGENAME eq LegacyStream.exe" # Check listening ports netstat -an | findstr ":8000\|:8001\|:8080"

Check Configuration

# Verify configuration file exists dir "C:\Program Files\LegacyStream\config\legacystream.ini" # Check configuration syntax type "C:\Program Files\LegacyStream\config\legacystream.ini"

Check Log Files

# View recent error logs type "C:\Program Files\LegacyStream\logs\error.log" # View server logs type "C:\Program Files\LegacyStream\logs\server.log"

1.2 Common Error Messages

  • "Port already in use": Another application is using the required port
  • "Access denied": Administrator privileges required
  • "Configuration file not found": Installation or file corruption issue
  • "SSL certificate error": Certificate configuration problem
  • "Audio codec not supported": Missing audio codec or format issue

2. Installation Issues

2.1 Installation Fails

Problem: Installation process fails or stops unexpectedly.
Solutions:
  1. Run installer as administrator
  2. Disable antivirus temporarily during installation
  3. Check available disk space (minimum 2 GB)
  4. Verify Windows version compatibility (Windows 10/11)
  5. Download installer again to check for corruption
# Check available disk space dir C:\ # Run installer as administrator Right-click installer â†’ "Run as administrator"

2.2 Missing Dependencies

Problem: Application fails to start due to missing DLLs or dependencies.
Solutions:
  1. Install Visual C++ Redistributable packages
  2. Install Qt runtime libraries
  3. Check PATH environment variable
  4. Reinstall LegacyStream
# Install Visual C++ Redistributable # Download from Microsoft website # Check PATH for Qt libraries echo %PATH%

2.3 Silent Installation Issues

Problem: Silent installation fails or doesn't complete properly.
Solutions:
  1. Use correct command line parameters
  2. Check for existing installations
  3. Run with administrator privileges
  4. Check installation logs
# Correct silent installation command LegacyStream-Setup.exe /S /D=C:\LegacyStream /L*V install.log # Check installation log type install.log

3. Startup Problems

3.1 Application Won't Start

Problem: LegacyStream application fails to launch or crashes immediately.
Solutions:
  1. Check Windows Event Viewer for error details
  2. Run application as administrator
  3. Check for conflicting applications
  4. Verify configuration file integrity
  5. Reinstall the application
# Check Windows Event Viewer eventvwr.msc # Run as administrator Right-click LegacyStream â†’ "Run as administrator" # Check for conflicting processes tasklist /FI "IMAGENAME eq LegacyStream.exe"

3.2 Port Binding Issues

Problem: Server fails to start due to port binding errors.
Solutions:
  1. Check if ports are already in use
  2. Change default ports in configuration
  3. Run as administrator for privileged ports
  4. Check firewall settings
# Check port usage netstat -an | findstr ":8000\|:8001\|:8080" # Kill process using port netstat -ano | findstr :8000 taskkill /PID [PID] /F

3.3 Configuration Errors

Problem: Application fails to start due to configuration file errors.
Solutions:
  1. Verify configuration file syntax
  2. Restore default configuration
  3. Check file permissions
  4. Validate configuration parameters
# Backup current configuration copy "C:\Program Files\LegacyStream\config\legacystream.ini" "legacystream.ini.backup" # Restore default configuration copy "C:\Program Files\LegacyStream\config\legacystream.ini.default" "legacystream.ini"

4. Streaming Issues

4.1 Streams Not Working

Problem: Audio streams fail to start or are not accessible.
Solutions:
  1. Check audio source configuration
  2. Verify mount point settings
  3. Test audio file formats
  4. Check stream logs
  5. Verify network connectivity
# Test stream URL curl http://localhost:8000/stream # Check stream logs type "C:\Program Files\LegacyStream\logs\stream.log"

4.2 Audio Quality Issues

Problem: Poor audio quality, distortion, or dropouts.
Solutions:
  1. Check audio bitrate settings
  2. Verify sample rate configuration
  3. Test different audio codecs
  4. Check system audio settings
  5. Monitor CPU and memory usage
# Check audio configuration [Audio] Bitrate=128 SampleRate=44100 Channels=2 Codec=mp3

4.3 Connection Problems

Problem: Clients cannot connect to streams or experience disconnections.
Solutions:
  1. Check firewall settings
  2. Verify network port configuration
  3. Test local and remote connections
  4. Check bandwidth limitations
  5. Monitor connection logs
# Test local connection telnet localhost 8000 # Check firewall rules netsh advfirewall firewall show rule name="LegacyStream"

5. Web Interface Problems

5.1 Web Interface Not Accessible

Problem: Cannot access the web interface at http://localhost:8080.
Solutions:
  1. Check if web server is running
  2. Verify port 8080 is not blocked
  3. Check firewall settings
  4. Try different browser
  5. Check web server logs
# Test web interface curl http://localhost:8080 # Check web server process tasklist /FI "IMAGENAME eq LegacyStream.exe"

5.2 Authentication Issues

Problem: Cannot login to web interface or authentication fails.
Solutions:
  1. Verify admin credentials
  2. Reset admin password
  3. Check authentication configuration
  4. Clear browser cache and cookies
  5. Try incognito/private browsing
# Reset admin password in configuration [Admin] Username=admin Password=new_password

5.3 Real-time Updates Not Working

Problem: Web interface doesn't show real-time updates or statistics.
Solutions:
  1. Check WebSocket connection
  2. Verify JavaScript is enabled
  3. Check browser console for errors
  4. Test with different browser
  5. Check WebSocket server logs
# Check WebSocket connection # Open browser developer tools (F12) # Look for WebSocket connection errors

6. GUI Issues

6.1 GUI Not Responding

Problem: GUI interface freezes or becomes unresponsive.
Solutions:
  1. Check system resources (CPU, memory)
  2. Restart the application
  3. Update graphics drivers
  4. Check for conflicting applications
  5. Run in compatibility mode
# Check system resources taskmgr # Kill unresponsive process taskkill /IM LegacyStream.exe /F

6.2 Theme Issues

Problem: GUI themes not working or display incorrectly.
Solutions:
  1. Reset to default theme
  2. Check theme file integrity
  3. Update Qt libraries
  4. Check display settings
  5. Reinstall application
# Reset theme configuration [GUI] Theme=default EnableCustomThemes=false

7. Network Problems

7.1 Firewall Issues

Problem: Network connections blocked by firewall.
Solutions:
  1. Add LegacyStream to firewall exceptions
  2. Configure specific port rules
  3. Check Windows Defender settings
  4. Test with firewall temporarily disabled
  5. Check third-party security software
# Add firewall rule netsh advfirewall firewall add rule name="LegacyStream" dir=in action=allow protocol=TCP localport=8000,8001,8080 # Check existing rules netsh advfirewall firewall show rule name="LegacyStream"

7.2 Port Conflicts

Problem: Ports already in use by other applications.
Solutions:
  1. Identify applications using ports
  2. Stop conflicting applications
  3. Change LegacyStream port configuration
  4. Use different port ranges
  5. Check for multiple LegacyStream instances
# Find process using port netstat -ano | findstr :8000 # Kill process by PID taskkill /PID [PID] /F

8. Security Issues

8.1 SSL Certificate Problems

Problem: SSL certificate errors or secure connections failing.
Solutions:
  1. Verify certificate file paths
  2. Check certificate validity
  3. Update certificate files
  4. Configure Let's Encrypt properly
  5. Test certificate with OpenSSL
# Test SSL certificate openssl s_client -connect localhost:8443 # Check certificate file openssl x509 -in certs/server.crt -text -noout

8.2 Authentication Failures

Problem: User authentication failing or access denied errors.
Solutions:
  1. Reset admin password
  2. Check user permissions
  3. Verify authentication configuration
  4. Check IP filtering settings
  5. Review security logs
# Reset admin password [Admin] Username=admin Password=new_secure_password # Check security logs type "C:\Program Files\LegacyStream\logs\security.log"

9. Performance Problems

9.1 High CPU Usage

Problem: LegacyStream using excessive CPU resources.
Solutions:
  1. Check audio encoding settings
  2. Reduce number of concurrent streams
  3. Optimize audio quality settings
  4. Check for background processes
  5. Update to latest version
# Monitor CPU usage tasklist /FI "IMAGENAME eq LegacyStream.exe" /FO TABLE # Check performance settings [Performance] MaxThreads=4 EnableOptimization=true

9.2 Memory Leaks

Problem: Application memory usage increasing over time.
Solutions:
  1. Restart application periodically
  2. Check for memory leaks in logs
  3. Update to latest version
  4. Monitor memory usage
  5. Configure memory limits
# Monitor memory usage wmic process where name="LegacyStream.exe" get WorkingSetSize # Set memory limits [Memory] MaxMemoryUsage=512MB EnableGarbageCollection=true

10. Audio Issues

10.1 No Audio Output

Problem: Streams have no audio or silent output.
Solutions:
  1. Check audio source configuration
  2. Verify audio file integrity
  3. Test different audio formats
  4. Check system audio settings
  5. Verify audio codec support
# Test audio file ffprobe audio_file.mp3 # Check audio configuration [Audio] InputDevice=default OutputDevice=default Volume=100

10.2 Audio Quality Problems

Problem: Poor audio quality, distortion, or artifacts.
Solutions:
  1. Increase audio bitrate
  2. Check sample rate settings
  3. Test different audio codecs
  4. Verify audio hardware
  5. Check for interference
# Optimize audio settings [Audio] Bitrate=192 SampleRate=48000 Channels=2 Codec=aac Quality=high

11. Log Analysis

11.1 Log File Locations

  • Server Log: logs/server.log
  • Error Log: logs/error.log
  • Access Log: logs/access.log
  • Debug Log: logs/debug.log
  • Security Log: logs/security.log

11.2 Common Log Messages

# Error messages to look for ERROR: Port 8000 already in use ERROR: SSL certificate not found ERROR: Audio codec not supported ERROR: Configuration file invalid WARNING: High memory usage detected INFO: Stream started successfully

11.3 Log Analysis Tools

# Search for errors findstr /i "error" logs\*.log # Search for warnings findstr /i "warning" logs\*.log # Monitor logs in real-time powershell "Get-Content logs\server.log -Wait"

12. Advanced Troubleshooting

12.1 Debug Mode

Enable debug mode for detailed troubleshooting:

# Enable debug logging [Logging] LogLevel=debug EnableDebugOutput=true LogToFile=true LogToConsole=true

Use built-in performance monitoring:

# Enable performance monitoring [Performance] EnableProfiling=true ProfileInterval=1000 LogPerformanceMetrics=true

12.4 System Diagnostics
