Requirements

Before installing LegacyStream, ensure your system meets the following requirements:

System Requirements

  • Operating System: Windows 7 or later (x64)
  • Processor: 1 GHz or faster
  • Memory: 2 GB RAM minimum (4 GB recommended)
  • Storage: 100 MB available space
  • Network: Stable internet connection

Development Environment (for building from source)

  • Visual Studio 2019 or later (with C++ workload)
  • CMake 3.20 or later
  • Qt 6.2 or later
  • OpenSSL 1.1.1 or later

Installation

Option 1: Download Pre-built Binary

  1. Download the latest release from the download section
  2. Extract the ZIP file to your desired installation directory
  3. Run LegacyStream.exe to start the application

Option 2: Build from Source

# Clone the repository
git clone https://github.com/yourorg/legacystream.git
cd legacystream

# Create build directory
mkdir build
cd build

# Configure with CMake
cmake .. -G "Visual Studio 17 2022" -A x64 -DCMAKE_BUILD_TYPE=Release

# Build the project
cmake --build . --config Release

# The executable will be in Release/LegacyStream.exe

Basic Configuration

Server Settings

The main configuration file is config.ini. Here's a basic example:

[Server]
HttpPort=8000
HttpsPort=8443
MaxConnections=100000
MaxStreams=1000
DefaultLatency=5

[SSL]
Enabled=false
AutoRenew=false

[WebInterface]
Enabled=true
UpdateInterval=1000
EnableRealTimeUpdates=true

[Streaming]
DefaultCodec=mp3
DefaultBitrate=128
BufferSize=65536

Mount Point Configuration

Mount points define where your streams are accessible. Example:

[MountPoints]
/live=icecast
/classic=shoutcast
/backup=icecast

[MountPoint:/live]
Name=Live Stream
Description=Main live broadcast
Codec=mp3
Bitrate=128
Quality=high
Public=true

Your First Stream

1. Create a Mount Point

In the GUI, go to the Streams tab and add a new mount point:

  • Mount: /live
  • Protocol: IceCast
  • Codec: MP3
  • Bitrate: 128 kbps

2. Configure Audio Source

Set up your audio input source:

  • Microphone input
  • Line-in audio
  • Virtual audio cable
  • Audio file playback

3. Start Broadcasting

Click the Start Stream button to begin broadcasting. Your stream will be available at:

  • HTTP: http://localhost:8000/live
  • HTTPS: https://localhost:8443/live

  • API Endpoints: http://localhost:8000/api/stats

Troubleshooting

Common Issues

Port Already in Use

Error: "Failed to bind to port 8000"

Solution: Change the port in settings or stop conflicting services

Audio Not Playing

Issue: Stream is running but no audio

Solution: Check audio input settings and ensure microphone access

High CPU Usage

Issue: Server using too much CPU

Solution: Reduce buffer sizes or adjust thread counts
