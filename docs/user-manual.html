1. Introduction

LegacyStream is a high-performance Windows-based audio streaming server with IceCast and SHOUTcast compatibility. It features a modern Qt-based GUI and comprehensive web interface for easy management of audio streams.

Key Features

  • Multi-Protocol Support: IceCast and SHOUTcast compatibility
  • Multi-Codec Support: MP3, AAC, OGG, FLAC, and more
  • Modern GUI: Qt-based desktop interface
  • Web Interface: Browser-based management
  • Real-time Monitoring: Live statistics and performance metrics
  • Security: SSL/TLS encryption and advanced security features
  • High Performance: Windows IOCP optimization

2. System Requirements

Minimum Requirements

  • Operating System: Windows 10 (64-bit) or later
  • Processor: Intel Core i3 or AMD equivalent
  • Memory: 4 GB RAM
  • Storage: 2 GB available disk space
  • Network: Broadband internet connection

Recommended Requirements

  • Operating System: Windows 11 (64-bit)
  • Processor: Intel Core i5 or AMD equivalent
  • Memory: 8 GB RAM or more
  • Storage: SSD with 10 GB available space
  • Network: High-speed internet connection
Note: LegacyStream requires administrator privileges for certain features like SSL certificate management and port binding.

3. Installation

3.1 Download

  1. Visit the official LegacyStream website
  2. Download the latest version for Windows
  3. Verify the download checksum for security

3.2 Installation Process

  1. Run the installer as administrator
  2. Accept the license agreement
  3. Choose installation directory (default: C:\Program Files\LegacyStream)
  4. Select components to install:
    • Core Server (required)
    • GUI Interface (recommended)
    • Web Interface (recommended)
    • Documentation (optional)
  5. Configure firewall settings
  6. Complete installation

3.3 Silent Installation

For automated deployment, use the silent installation option:

LegacyStream-Setup.exe /S /D=C:\LegacyStream
Tip: The installer will automatically configure Windows Firewall rules for the required ports.

4. Initial Setup

4.1 First Launch

  1. Launch LegacyStream from the Start Menu or desktop shortcut
  2. The setup wizard will guide you through initial configuration
  3. Configure basic server settings:
    • Server name and description
    • Admin username and password
    • Default ports (8000 for IceCast, 8001 for SHOUTcast)
    • Stream directory location
  4. Test the configuration
  5. Complete setup

4.2 Configuration File

The main configuration file is located at:

C:\Program Files\LegacyStream\config\legacystream.ini

4.3 Default Settings

  • Admin Username: admin
  • Default Password: (set during installation)
  • IceCast Port: 8000
  • SHOUTcast Port: 8001
  • Web Interface Port: 8080

5. Configuration

5.1 Server Configuration

Edit the main configuration file to customize server settings:

[Server] Name=My LegacyStream Server Description=High-quality audio streaming AdminUsername=admin AdminPassword=your_secure_password [Network] IceCastPort=8000 ShoutcastPort=8001 WebInterfacePort=8080 MaxConnections=1000 [Security] EnableSSL=true SSLCertificatePath=certs/server.crt SSLPrivateKeyPath=certs/server.key

5.2 Mount Point Configuration

Configure mount points for your audio streams:

[MountPoints] /live=Live Stream /music=Music Stream /news=News Stream [MountPoint:/live] Type=live Format=mp3 Bitrate=128 Channels=2 SampleRate=44100

5.3 SSL/TLS Configuration

For secure streaming, configure SSL certificates:

  1. Obtain SSL certificates (Let's Encrypt recommended)
  2. Place certificates in the certs directory
  3. Update configuration with certificate paths
  4. Enable SSL in the configuration
  5. Restart the server

8. Streaming

8.1 Creating Streams

  1. Configure audio source (file, microphone, or external input)
  2. Set audio format and quality parameters
  3. Choose mount point path
  4. Configure metadata settings
  5. Start the stream

8.2 Supported Formats

  • Input Formats: MP3, AAC, OGG, FLAC, WAV
  • Output Formats: MP3, AAC, OGG, FLAC
  • Bitrates: 32 kbps to 320 kbps
  • Sample Rates: 22050 Hz to 48000 Hz
  • Channels: Mono and Stereo

8.3 Stream URLs

Once streams are created, they can be accessed via:

  • IceCast: http://your-server:8000/mountpoint
  • SHOUTcast: http://your-server:8001/mountpoint
  • SSL: https://your-server:8443/mountpoint

8.4 Metadata

Configure stream metadata for better listener experience:

[Metadata] Title=My Radio Station Artist=Various Artists Genre=Pop Description=Best music 24/7 URL=http://myradio.com

9. Security

9.1 SSL/TLS Configuration

  1. Obtain SSL certificates from a trusted CA
  2. Install certificates in the certs directory
  3. Update configuration with certificate paths
  4. Enable SSL in server settings
  5. Test secure connections

9.2 Access Control

  • Admin Authentication: Secure admin login
  • Source Authentication: Protect stream sources
  • IP Filtering: Restrict access by IP address
  • Rate Limiting: Prevent abuse

9.3 Firewall Configuration

Configure your firewall to allow LegacyStream traffic:

  • TCP Port 8000: IceCast streaming
  • TCP Port 8001: SHOUTcast streaming
  • TCP Port 8080: Web interface
  • TCP Port 8443: SSL streaming (if enabled)
Security Note: Always use strong passwords and keep your system updated.

11. Troubleshooting

11.1 Common Issues

Server Won't Start

  • Check if ports are already in use
  • Verify configuration file syntax
  • Check administrator privileges
  • Review error logs

Streams Not Working

  • Verify audio source configuration
  • Check mount point settings
  • Test audio file formats
  • Review stream logs

Web Interface Not Accessible

  • Check if web server is running
  • Verify port 8080 is open
  • Check firewall settings
  • Test with different browser

11.2 Log Analysis

Use log files to diagnose problems:

# View recent server logs tail -f logs/server.log # Check for errors grep ERROR logs/error.log # Monitor access logs tail -f logs/access.log
